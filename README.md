## Features

- Play Tic-Tac-Toe against an AI that can't be beaten.
- The AI's logic is based on the minimax algorithm, a classic decision-making algorithm in game theory.
- Alpha-beta pruning is implemented for optimization, making the AI's decision process more efficient.
